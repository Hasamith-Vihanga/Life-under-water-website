Splash Page

Technical Description

The splash page has been built in HTML for structural purposes, employed CSS to give visual styling effects, and animation in order to create an engaging, vividly beautiful diving underwater-themed introduction screen.

Accessibility

Ensured by high contrasts text, utilizing semantic HTML elements, as well as an automatic redirect notification, all can read, make use of, and be readable by screen readers.

Volunteer Page

Technical Description

The Volunteer page adheres to an HTML structure with CSS applied for styling, allowing it to be responsive, legible, and enjoyable.

Accessibility

Heading hierarchy, images with proper alt text, and proper color contrast improve usability for a variety of users, including those relying on a screen reader. Keyboard navigability is also provided.

Content Page

Technical Description

The content page has an HTML-based structure and CSS-based design with responsiveness, readability, and aesthetic appeal of underwater themes.

Accessibility

We have leveraged the use of semantic HTML, provided alt text to images, created clear and readable headings, used high color contrasts, and enabled the content to be navigated via the keyboard-a full-fledged approach towards accessibility.
